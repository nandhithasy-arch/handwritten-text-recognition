# Handwritten Text Recognition System

An AI-based web app that converts handwritten images into editable text using Gradio and TrOCR.

## Features
- Upload handwritten images.
- Convert handwriting to text.
- Simple and interactive Gradio UI.
- Built with Python, OpenCV, and Hugging Face Transformers.

## Tech Stack
- Python
- Gradio
- TrOCR
- OpenCV
- Pillow
- NumPy

## Installation
```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
pip install -r requirements.txt
```

## Run the App
```bash
python app.py
```

## Usage
1. Open the app.
2. Upload a handwritten image.
3. Click the convert button.
4. Read the extracted text.
